# To-Do List App


## 🚀 스택

- `react`
- `javaScript`
- `SCSS`(SASS)
- `classnames`, `react-icons`, etc.


## Setting

``` bash
$ git clone https://github.com/thisisyjin/toDoList.git

$ cd toDoList
$ npm install

$ npm start || yarn start
```
